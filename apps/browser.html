Features

Rethinking the daily browser

⚡

Chromium Base

Runs on Electron's robust Chromium webview for full compatibility with modern websites.

🎨

Custom Themes

Comes with built-in custom themes. Transition seamlessly from a standard view to attic or dark modes.

🔒

Resource Efficient

Uses optimized garbage collection and preload configurations to keep memory usage lower than generic browsers.

⚙️

Preload Scripts

Includes preloaded features and isolated contexts for secure execution and background services.
