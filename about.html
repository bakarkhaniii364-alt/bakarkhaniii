About

Building things
worth building.

Bakarkhaniii is a personal dev hub. A collection of native apps, browser extensions, browser games, and webapps built by one person who cares about quality, speed, and doing things properly.

The name sounds like a lot. That's intentional. There's a lot here — and more coming. Every project under this roof is built to the same standard: clean design, focused features, and software that respects the person using it.

The products tackle real friction. Trackxpense came from being annoyed at spreadsheets every month. Gupto came from building a fun, engaging word deduction game to play with friends. Both came from building things I actually wanted to use.

The extensions — AEGIS and Unfollow Ext — came from similar utility. Speeding up search actions and managing Instagram followers directly and safely within the browser.

This is a long-term project. Every piece gets refined, not abandoned. If you find a bug, want a feature, or just want to talk — reach out.

Values

How everything here gets built

🎯 Focused

Every product does one thing well. No bloat, no feature creep, no dark patterns. If a feature doesn't serve the core use case, it doesn't ship.

🔒 Private by default

Your data is yours. We collect as little as possible, store nothing we don't need, and always give you a way out.

⚡ Fast, always

Performance is a feature. Every page loads fast. Every app responds instantly. Slow software is disrespectful of people's time.

🛠️ Built to last

Nothing here is a prototype left to die. Every project is maintained, improved, and treated like a real product — because it is.
